# Video Background

Place your video background file here named `travel-bg.mp4`.

## Requirements:
- **Filename**: Must be named `travel-bg.mp4`
- **Format**: MP4 video format
- **Recommended specs**:
  - Resolution: 1920x1080 (Full HD) or 1280x720 (HD)
  - File size: Under 10MB for best performance
  - Duration: 10-30 seconds (will loop automatically)
  - No audio required (video will be muted)

## Where the video is used:
1. **Login Page**: Full-screen background for the login interface
2. **Chat Windows**: Background for all 6 multi-agent chat interfaces
   - Flights Agent
   - Hotels Agent
   - Rides Agent
   - Nearby Attractions Agent
   - AI Agent
   - Trips Agent

## Finding a video:
You can find free travel-themed videos from:
- **Pexels**: https://www.pexels.com/search/videos/travel/
- **Pixabay**: https://pixabay.com/videos/search/travel/
- **Unsplash**: https://unsplash.com/s/videos/travel

## Tips:
- Choose videos with smooth, slow motion for best effect
- Prefer videos with natural landscapes or travel destinations
- Avoid videos with too much movement or text overlays
- The video will have an overlay applied for better text readability
